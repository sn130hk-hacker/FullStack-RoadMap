6. Key CSS Concepts Used

CSS Variables (Custom Properties)

Define reusable values: :root { --primary-color: #3498db; }

Flexbox

One-dimensional layout: display: flex;

Grid

Two-dimensional layout: display: grid;

Transitions

Smooth animations: transition: all 0.3s ease;

Transforms

Move elements: transform: translateY(-2px);

Media Queries

Responsive design: @media (max-width: 768px) { ... }

Box Shadow

Add depth: box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
